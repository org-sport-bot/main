# @org_sport_bot

> 🤖 Track activities, manage participants, and monitor payments — all inside Telegram.  
> No schedules, no reminders, just the essentials.

🔗 **Launch bot**: [t.me/org_sport_bot](https://t.me/org_sport_bot)

---

## ⚠️ Important Notice

- 🔐 **Do not share personal data** (phone, email, ID numbers) — the bot is not designed for PII storage.
- 📦 All data is provided **"as is"**. No backups, no recovery, and no liability for data loss.
- ⚖️ Use at your own risk. This is a lightweight utility, not a certified accounting system.

## 🚀 Quick Start

1. Open Telegram and find **@org_sport_bot**
2. Tap `/start` and choose your role: **Organizer** or **Participant**
3. Create your first activity or register as a participant
